# health-iot

这是一个基于MQTT协议的物联网健康监测系统，目前可以监测人体的心率、血氧等生理参数；
也可以监测家庭的温湿度、烟雾浓度等环境参数

#### 2.Software\IOT-App\pages：微信小程序各个页面及功能
